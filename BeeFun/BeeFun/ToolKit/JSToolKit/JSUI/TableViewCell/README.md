

# 配置数据源

Plist/JSON

Plist针对TableView的结构仅支持，如下：

    <array>
        <array>                 ------------------Section 1
            <dictionary>                    ------------Section 1 Row 1
                [type:label]
                [key:Language]
                [value:En]
            </dictionary>
            <dictionary>                    ------------Section 1 Row 2
                [type:label]
                [key:Language]
                [value:En]
            </dictionary>
        </array>
    
        <array>                 ------------------Section 2
            <dictionary>                    ------------Section 2 Row 1
                [type:iamge]
                [key:About]
                [icon:logo.png]
            </dictionary>
        </array>
    </array>
